ЗАГРУЗКА РАСПИСАНИЯ В БД:

ВСЕ БУКВЫ КЛАССА НЕОБХОДИМО ПИСАТЬ КИРИЛИЦЕЙ!!!


add_schedules() - Принимает на вход {"Класс": {"День недели" : {Номер урока : "Название урока", ...
                  {"5A": {"Понедельник" : {0: Литература, ...

ВЫГРУЗКА РАСПИАНИЯ ИЗ БД:

get_schedule() - Принимает строку с названием класса: "7A"
                 Возвращает: {"День недели" : {'Номер урока' : ('id урока', "Название урока", Номер урока, id расписания, "День недели", Номер кабинета, Имя учителя, Дата урока ...

get_schedules() - Принимает массив с названиями классов: ["7А", "9В" ...
                  Возвращает: {"Номер класса" : {"День недели" : {'Номер урока' : ('id урока', "Название урока", Номер урока, id расписания, "День недели", Номер кабинета, Имя учителя, Дата урока ...

get_schedule_by_date() - Принимает название класса ("7А", "9В") и дату. Если дата не введена, будет возвращено
    сегодняшнее расписание. Можно передать дату в формате YYYYMMDD. А можно задать date=cur_date(add),
    где add - кол-во дней смещения от сегодняшней даты. (например, cur_date(1) - завтрашняя дата)
